Boehm garbage collector
=======================
- https://www.hboehm.info/gc/
- https://github.com/bdwgc/bdwgc

Install
-------
`% sudo pkg install boehm-gc boehm-gc-threaded`

pkgconf
-------
```sh
% ls /usr/local/libdata/pkgconfig/bdw-gc*
/usr/local/libdata/pkgconfig/bdw-gc-threaded.pc
/usr/local/libdata/pkgconfig/bdw-gc.pc
```
